This folder contains the code for the API used for the [M3U8 list](/public/). It replicates the requests made by [the website's `special://` protocol](https://github.com/ESCFent/UnofficialESCBroadcasts/blob/main/script.js#L57).

It was made with [Deno](https://bun.com) and runs on [Alwaysdata](https://alwaysdata.com) at [api.escfent.com](https://api.escfent.com).
